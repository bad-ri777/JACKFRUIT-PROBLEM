# JACKFRUIT-PROBLEM
A simple OCR-powered tool that digitizes handwritten shop records and turns them into searchable data. Our project helps small shop owners view summaries, generate charts, and manage their daily numbers with ease.

### Image Input and Processing
* **`read_table_from_image.py`**: Given an image input, it performs geometric alignment, detects table borders, crops each cell and saves them to a folder named "cells" for further processing.
